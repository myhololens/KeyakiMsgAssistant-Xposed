# KeyakiMsgAssistant
Xposed module for "欅坂46メッセー" app
(Without user interface)

## Requirement
- [x] Android 4.1+
- [x] Rooted
- [x] Xposed framework

## Permission
- INTERNET
- WRITE_EXTERNAL_STORAGE
- SYSTEM_ALERT_WINDOW

## Feature
- Add long-press-to-copy function for text
- Help to download photos, audios, videos in messages
